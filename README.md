# Introduction

A very simple example to use react-redux, using following libraries:

- react
- redux
- react-redux
- redux-thunk
- redux-devtools
- fetch API
- webpack
- express


# Development

1. Run `npm install`
2. Run `npm run dev`
3. Open browser and visit **[http://localhost:3000/](http://localhost:3000/)**


For **production**, please run `npm run build`
